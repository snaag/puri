## 팀원
* `Frontend` 김민재, 손명균
* `Backend` 이상아, 홍진영

### 작업 내용
* 이상아
   - Sequelize를 사용하여, RDS에 서버 구축
   - EC2에 서버 배포
   - REST API를 따르도록 구현
   - 회원정보 CRU, 문제 CRUD 구현
